Proyectos realizados por Juan Manuel Godoy

Proyecto CineCinco

Este proyecto fue desarrollado a fin de aplicar los conocimientos adquiridos en lenguajes de programación referidos al frontend como Thymeleaf, CSS y boostrap.

El mismo se refiere a una aplicación destinada a la publicación de la cartelera de un cine ficticio, como también para la venta de entradas y el cobro de las mismas a través de un plataforma digital de pagos.

Proyecto Meetmedia

Este proyecto fue desarrollado para la aplicación de los conocimientos obtenidos en el Curso de Desarrollo Full Stack en Egg Educación; el mismo fue presentado a modo de trabajo final. Se aplicaron los lenguajes estudiados en esta Institución: Java, HTML y Mysql
